# 简约单页导航系统带后台

## 系统介绍
navigation-page是一个简约风格的单页导航系统，在原系统的基础上添加了系统后台管理界面，所有数据管理通过MySQL数据库完成，使用PHP+MySQL编写。

## 数据库结构
- `category` 表：存储分类信息
- `data` 表：存储导航数据

## 安装使用
1. 初次使用需要将 `nav.sql` 文件导入MySQL数据库
2. 在 `index.php` 页面修改数据库配置信息
3. `bg.jpg` 首页背景横幅可以自行替换修改
